## Common Commands

- `npm run build` builds responsive images, CSS, and JS.
- `npm run stage:pages` builds and stages the deployable site into `.deploy/public`.
- `npm run check:local` checks staged HTML and local site references.
- `npm run audit:externals` audits external links and mirrored media.
- `npm run preview` stages the site and serves `.deploy/public` locally.
